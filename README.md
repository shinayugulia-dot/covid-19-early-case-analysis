# COVID-19 Early Case Trend Analysis & Recovery Insights

## Project Overview
This project analyzes early COVID-19 patient data to understand infection patterns,
demographic trends, recovery duration, and factors influencing recovery outcomes.

## Objectives
- Analyze demographic patterns of infected patients
- Study recovery trends and timelines
- Compare regional impact of COVID-19
- Identify factors influencing recovery duration using regression

## Methodology
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Visualization of trends
- Correlation analysis
- Linear regression modeling

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Dataset
- Primary: Patient-level COVID-19 data
- Fallback: Simulated dataset used when source data is unavailable
